# Multi-Layer Plagiarism Detection

A comprehensive plagiarism detection system that uses multiple layers of analysis to identify potential plagiarism in academic and professional documents.

## 🎯 Overview

This project implements a sophisticated plagiarism detection system combining JavaScript, Python, and CSS technologies. It utilizes multiple detection algorithms and comparison techniques across different layers to provide accurate and reliable plagiarism identification.

- **Total Language Composition:**
  - JavaScript: 44.8%

## ✨ Features

- **Multi-Layer Detection**: Implements multiple plagiarism detection algorithms
- **Document Analysis**: Analyzes text structure, content, and patterns
- **Comparison Engine**: Sophisticated document comparison capabilities
- **User-Friendly Interface**: Clean and intuitive web-based interface
- **Detailed Reports**: Comprehensive plagiarism detection reports with similarity scores
- **Fast Processing**: Optimized for quick document analysis

## 🚀 Quick Start

### Installation

1. Clone the repository:
```bash
git clone https://github.com/praveen1-tech/multi-layer-plagiarism-detection.git
cd multi-layer-plagiarism-detection
```

2. Install Python dependencies:
```bash
pip install -r requirements.txt
```

3. Install Node.js dependencies:
```bash
npm install
```

### Running the Application

1. Start the backend server:
```bash
python app.py
```

2. Start the frontend development server:
```bash
npm start
```

3. Open your browser and navigate to `http://localhost:3000`

## 📚 Usage

1. Upload documents to compare
2. Select the detection layer(s) you want to use
3. Initiate the plagiarism detection analysis
4. Review the detailed plagiarism report with similarity scores
5. Export the report as needed

## 🔍 Detection Layers

The system implements multiple detection layers:
- **Textual Similarity**: Analyzes text patterns and word frequencies
- **Structural Analysis**: Examines document structure and formatting
- **Semantic Analysis**: Compares meaning and content similarity
- **Metadata Analysis**: Reviews document metadata and properties
